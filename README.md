# Spring Boot Book REST API

This project is a simple REST API built using Spring Boot.

## Features
- Get all books
- Get book by ID
- Add new book
- Update book
- Delete book

## Technologies Used
- Java
- Spring Boot
- REST API
- Postman
- HashMap (In-memory storage)

## API Endpoints

GET /books  
POST /books  
PUT /books/{id}  
DELETE /books/{id}

## Testing

All APIs were tested using Postman.

## Project Structure

Controller → Service → Repository → Model
